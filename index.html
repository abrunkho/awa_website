About the Experience

Built by scientists and educators, Are We Alone? is a web-based learning experience that integrates virtual tours built with NASA data and imagery with personalized learning pathways supported by adaptive feedback.


But it’s more than just an experience. It’s a framework that can expand and grow with new discoveries and contributions from scientists like you!
